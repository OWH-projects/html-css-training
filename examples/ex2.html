Our CSS rules make h2 heds red

and paragraphs bold ...

... unless you override their styling. An example of making inline style rules on the fly.

You can create a CSS class and apply it to an HTML element, like we did with this div here. (Think of <div> tags as rooms in a home -- it's a way to compartmentalize content to target for styling or scripting.) On this page, all HTML elements with the class "banana" will have black backgrounds with white text, 20 pixels of padding and 2 ems of top margin, and they'll fill 50 percent of the screen width.

You can also target an element by its unique id. Here, the element called "chihuahua" is given an opacity of 20 percent.
